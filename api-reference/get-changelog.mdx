---
title: 'Get Change Log'
api: 'GET /api/external/change-log'
description: 'Fetch platform change log entries'
---

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your WURK API key
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

## Response

<ResponseField name="entries" type="array">
  Array of change log entries
</ResponseField>

<ResponseField name="page" type="integer">
  Current page number
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages
</ResponseField>

<ResponseField name="pageSize" type="integer">
  Number of items per page (50)
</ResponseField>

## Rate Limit

5 requests per minute

## Caching

Server caches each page for 1 hour

<RequestExample>

```bash
curl -X GET "https://wurkapi.fun/api/external/change-log?page=1" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Accept: application/json"
```

</RequestExample>
