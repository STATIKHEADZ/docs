---
title: 'Get Job Submissions'
api: 'GET /api/external/jobs/{jobId}/submissions'
description: 'List submissions for a custom job using the job ID'
---

## Authentication

<ParamField header="X-API-Key" type="string" required>
  Your WURK API key
</ParamField>

## Path Parameters

<ParamField path="jobId" type="string" required>
  The job ID (8 character identifier like `f80a0038`)
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

## Response

<ResponseField name="submissions" type="array">
  Array of submissions with the following fields:
  - `id`: Submission ID
  - `content_text`: Submission text
  - `attachment_urls`: Array of attachment URLs (if any)
  - `winner`: Winner status (0 = not winner, 1 = winner)
</ResponseField>

<ResponseField name="page" type="integer">
  Current page number
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages
</ResponseField>

<ResponseField name="pageSize" type="integer">
  Number of items per page (50)
</ResponseField>

## Rate Limit

6 requests per minute

## Requirements

- Job must be of type `custom`
- Job must belong to your API key

<Note>
  To test this endpoint, you need a valid job ID from a custom job that belongs to your API key. 
  First create a custom job using the [Create Custom Job](/api-reference/create-custom-job) endpoint, 
  then use the returned `jobId` (not `customId`) here.
</Note>

<Warning>
  **Try It Issues**: The "Try it" feature may not work correctly with path parameters. 
  If you get "job not found" in the Try popup but the job exists:
  - Use the curl command directly in your terminal
  - Make sure to replace `{jobId}` with your actual job ID (e.g., `f80a0038`)
  - Ensure you're using the correct API key that owns the job
</Warning>

<RequestExample>

```bash
curl -X GET "https://wurkapi.fun/api/external/jobs/{jobId}/submissions?page=1" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Accept: application/json"
```

```bash Working Example
# This is a real job that exists and has submissions
curl -X GET "https://wurkapi.fun/api/external/jobs/f80a0038/submissions?page=1" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Accept: application/json"
```

</RequestExample>
